# The Lumen Care

The Lumen Care is a premium personal care brand focused on high-performance solid shampoo bars designed for scalp health, hair strength, and sustainability.

## 🚀 Overview

This repository contains:
- Product formulations (solid shampoo bars)
- Ingredient research and documentation
- Manufacturing process guidelines
- Quality control standards
- Cost optimization models

## 🧪 Core Product

**Solid Shampoo Bar**
- Sulfate-free surfactant system (SCI-based)
- Conditioning agents (BTMS, Cetearyl Alcohol)
- Nourishing oils (Shea Butter, MCT Oil)
- Protein enrichment (Hydrolyzed Silk)

## ⚙️ Technical Approach

- Optimized surfactant ratios for cleansing vs mildness
- Structured fatty alcohol system for bar hardness
- pH-balanced formulations (target: 5.0–5.5)
- Low water activity for longer shelf life
